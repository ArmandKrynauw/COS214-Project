# Setup
## Prerequisites

* CMake >= 3.2

## Cloning
```
git clone 
```
## Building
This project can be build using CMake:
```
mkdir build
cd build
cmake ..
make
```

## Running
For Web UI mode:
```
make run
```
* Go to src/GUI/splashPage/
* Open splash.html

For design mode:
```
make dev
```

# Test
## Prerequisites
* CMake >= 3.2
* libgtest

## Building
```
cd Test
mkdir build
cd build
cmake ..
make
```

## Running
```
./runTests
```
